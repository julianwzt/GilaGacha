# GilaGacha

mini-game kompetitif berbasis web (HTML, CSS, JS) bergaya arcade-battle dengan tema visual neon futuristik. Dalam game ini, adu mekanik timing dan keberuntungan digabung menjadi satu! Dua pemain akan saling berhadapan menggunakan mesin slot 3D untuk memperebutkan damage tertinggi dan menghancurkan HP lawan.

Fitur Utama:

Mekanik Spin 2 Fase: Pemain harus menekan "STOP" di momen yang tepat untuk mengunci nilai dasar (POWER) dan angka pengali (EXTRA). Hati-hati, ada zonk (MISS)!

Clash Animation: Angka total dari kedua pemain akan terbang dan bertabrakan di tengah arena. Layar akan bergetar dan menghasilkan efek suara ledakan jika ada yang terkena hit.

Sistem Turnamen: Permainan tidak selesai dalam satu ronde. Pemain harus mengumpulkan 3 Mahkota (Crown) untuk benar-benar keluar sebagai "JUARA MATCH".

Fitur Mega Jackpot (Instant K.O): Jika pemain super hoki mendapatkan kombinasi maksimal (500 x 3.0), game akan memicu animasi brutal dengan layar redup, getaran hebat, dan suara ledakan epik yang langsung membunuh lawan dalam satu serangan (1500 Damage).

Lawan Teman atau Bot: Bisa dimainkan berdua secara lokal di satu perangkat, atau bermain mode "Sendiri" melawan Bot yang dirancang untuk menekan tombol secara otomatis.

Kustomisasi Tema Neon: Pemain dapat mempersonalisasi warna neon mereka sendiri (Cyan, Hijau, Kuning, Oranye, Pink, atau Ungu), yang akan mengubah seluruh efek cahaya, HP bar, dan nama mereka di dalam arena.

Responsif: Bisa dimainkan dengan mulus dan proporsional baik di layar lebar (PC/Laptop) maupun di layar smartphone (Mobile).
